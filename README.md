# 宠物之家 - 全栈宠物服务平台

一个现代化的宠物服务平台，提供宠物商品购买、服务预约、用户管理等功能。

## 🚀 项目特色

- **完整的电商功能**：商品浏览、购物车、订单管理、支付流程
- **宠物服务预约**：在线预约宠物洗护、美容等服务
- **用户中心**：个人信息管理、订单查看、收藏管理
- **现代化UI设计**：响应式设计，支持移动端和桌面端
- **完整的后端API**：Spring Boot + MyBatis 构建的RESTful API

## 🛠️ 技术栈

### 前端
- **Vue 3** - 渐进式JavaScript框架
- **Vue Router** - 官方路由管理器
- **Pinia** - 状态管理
- **Vite** - 现代化构建工具
- **Iconify** - 图标库

### 后端
- **Spring Boot 3** - Java企业级应用框架
- **MyBatis** - 持久层框架
- **MySQL** - 关系型数据库
- **JWT** - 身份认证
- **Maven** - 项目管理工具

## 🚀 快速开始

### 环境要求
- Node.js 16+
- Java 17+
- MySQL 8.0+
- Maven 3.6+

### 前端启动
```bash
cd my-pet-platform
npm install
npm run dev
```
访问：http://localhost:5173

### 后端启动
```bash
cd backend
mvn spring-boot:run
```
访问：http://localhost:8080

### 数据库配置
1. 创建MySQL数据库 `pet_home`
2. 导入测试数据（pet_products_data目录）
3. 修改 `backend/src/main/resources/application.yml` 中的数据库配置

## 📱 页面展示

### 主要页面
- **首页** - 商品展示和导航
- **商品详情** - 详细的商品信息和购买选项
- **购物车** - 商品管理和结算
- **结算页面** - 地址选择、支付方式、订单确认
- **用户中心** - 订单管理、个人设置、收藏等
- **服务预约** - 宠物服务在线预约

### 核心功能流程

#### 1. 购物流程
```
浏览商品 → 查看详情 → 加入购物车/立即购买 → 选择地址 → 确认订单 → 扫码支付 → 订单完成
```

#### 2. 预约流程
```
选择服务 → 填写宠物信息 → 选择时间 → 确认预约 → 服务完成
```

#### 3. 用户管理
```
注册登录 → 完善个人信息 → 添加宠物档案 → 管理收货地址 → 设置账户偏好
```

## 🔧 开发说明

### API接口
- 基础URL：`http://localhost:8080/api`
- 认证方式：JWT Token
- 数据格式：JSON

### 主要接口
- `/api/auth/*` - 用户认证
- `/api/products/*` - 商品管理
- `/api/cart/*` - 购物车
- `/api/orders/*` - 订单管理
- `/api/appointments/*` - 服务预约
- `/api/addresses/*` - 地址管理
- `/api/favorites/*` - 收藏管理
